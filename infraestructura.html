Home-Lab Architecture

Administración de Sistemas

Administración de Ubuntu Server sobre hardware dedicado (Intel i5 8th Gen). Mi enfoque principal es la disponibilidad crítica: utilizo Docker Compose para orquestar microservicios que soportan mis experimentos de IA y almacenamiento persistente vía Samba.

Visión de Continuidad

Diseño de arquitectura de almacenamiento basado en RAID 5+1 para garantizar integridad ante fallos de disco. Implementación de respaldos diferenciales y acceso remoto seguro mediante redes VPN mesh (Tailscale).
