BIOTRACKER

IA Vision & Pipeline de Datos

Desafío de Ingeniería

El proyecto BioTracker surge para automatizar la identificación de especies en entornos naturales. El reto técnico principal es la transformación de datos visuales desestructurados en una base de datos científica coherente.

Metodología de Desarrollo

  • • Preprocesamiento: Normalización de datos con Scikit-learn para reducir el sesgo ambiental.
  • • Inferencia: Implementación de redes neuronales con TensorFlow v2.x.
  • • Persistencia: Integración híbrida SQL (Integridad) y NoSQL (Velocidad).
