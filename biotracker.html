BioTracker

Fase: Diseño e Integración de Datos

Descripción del Proyecto

BioTracker nace como una solución a la necesidad de monitorear fauna en tiempo real. Utilizando modelos de visión por computadora, el sistema identifica especies y genera registros automáticos.

Reto en Bases de Datos

Actualmente trabajo en la arquitectura híbrida: usando SQL para la consistencia de catálogos taxonómicos y NoSQL para la alta velocidad de ingesta de detecciones visuales.

Stack Técnico

  • • Python (OpenCV)
  • • MariaDB / NoSQL (Modeling)
  • • Data Normalization
