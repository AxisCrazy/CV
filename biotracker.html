BIOTRACKER

Integración de ML & Deep Learning

Metodología Técnica

El proyecto utiliza TensorFlow para el entrenamiento de redes neuronales de visión y Scikit-learn para el preprocesamiento de características y modelos de clasificación estadística. La meta es crear un pipeline automatizado de ingesta de datos de campo.

Flujo de Datos

Preprocesamiento

Limpieza de ruido y normalización de variables ambientales mediante Scikit-learn.

Inferencia

Identificación de patrones biológicos complejos con modelos de Deep Learning.

IA Library Stack

  • TensorFlow v2.x
  • Scikit-learn Applied
  • OpenCV In Use
