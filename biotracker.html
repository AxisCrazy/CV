BIOTRACKER

Ingeniería de IA & Pipelines de Datos

Arquitectura del Proyecto

BioTracker automatiza la identificación de fauna mediante visión artificial. El núcleo técnico reside en la capacidad de procesar flujos de video y convertirlos en metadatos estructurados para investigación científica.

Visión (Deep Learning)

Uso de TensorFlow para el entrenamiento de redes neuronales convolucionales enfocadas en la clasificación de especies.

Procesamiento de Datos

Implementación de Scikit-learn para la normalización de variables ambientales y limpieza de ruido en el dataset.

Estrategia de Almacenamiento

Se implementó una solución híbrida para maximizar la eficiencia:

  • SQL (MariaDB): Manejo de integridad referencial para taxonomía y registros de avistamientos validados.

  • NoSQL (En proceso): Estructura de logs de alta velocidad para capturas de metadatos temporales y datos no estructurados.

Ficha Técnica

  • EntornoPython 3.x
  • ModeladoTF / SKLearn
  • Data FlowETL Pipeline
