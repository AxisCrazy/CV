BIOTRACKER

ML & Deep Learning Pipeline

Metodología Técnica

El proyecto utiliza TensorFlow para el entrenamiento de redes neuronales de visión y Scikit-learn para el preprocesamiento de características y modelos de clasificación. La meta es crear un pipeline automatizado de ingesta de datos de campo.

Arquitectura de Datos

Diseño híbrido: SQL (MariaDB) para registros taxonómicos y NoSQL para logs de avistamiento de alta velocidad y metadatos no estructurados.

Tech Stack

  • TensorFlow Applied
  • Scikit-learn Pre-process
  • OpenCV CV Task
